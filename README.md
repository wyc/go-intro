go-intro
-------------

To build the presentation:

1. Install [pandoc](http://pandoc.org/) and [LaTeX](https://www.latex-project.org/get/).
2. Run the Makefile:
```
$ make
pandoc -t beamer -o go-intro.pdf go-intro.md
```
